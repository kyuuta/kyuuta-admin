## 简介

`KYUUTA Admin` 一个基于vue2+,antdv构建的管理端

## 目标

做一个好用、满足大部分需求的后台管理端

## Build Setup
``` bash
# clone the project
git clone https://github.com/kyuuta/kyuuta-admin.git
cb kyuuta-admin

# install dependencies
npm install

# serve with hot reload at localhost:23333
npm run serve
```
