## 特性

### 简单易用
使用 TypeScript 构建类型系统，尽可能提供类型提示。封装了管理系统的大部分常用组件。

### 灵活配置
尽可能的支持配置项修改.

### 可学性
使用了unocss、vue3、naiveui、pinia等较新的技术实现，尽可能的提供最优解决方案与开源社区共同学习,也欢迎提交PR&ISSUES.

## 📦 安装构建
```bash
# 安装依赖
pnpm install

# 启动项目
npm run serve
```
